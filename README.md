# Реализация утилит cat и grep на языке C

Данный проект представляет собой реализации двух классических утилит командной строки — `cat` и `grep` — написанные на языке C в образовательных целях. Утилиты поддерживают основные опции, аналогичные стандартным версиям в Unix-подобных системах.


## Обзор

Проект состоит из двух независимых подпроектов:
- **cat** — утилита для конкатенации и вывода содержимого файлов с поддержкой различных флагов форматирования.
- **grep** — утилита для поиска текста по шаблону в файлах с поддержкой регулярных выражений и множества опций.

Обе утилиты разработаны с соблюдением стандартов C11 и компилируются с флагами `-Wall -Werror -Wextra` для обеспечения высокого качества кода.

## Возможности

### cat

Реализация `cat` поддерживает следующие возможности:
- Вывод содержимого одного или нескольких файлов.
- Нумерация строк:
  - `-n` — нумерация всех строк.
  - `-b` — нумерация только непустых строк (игнорирует пустые строки).
- Сжатие пустых строк (`-s`) — несколько подряд идущих пустых строк выводятся как одна.
- Отображение непечатаемых символов:
  - `-v` — отображение непечатаемых символов в виде `^X` (для управляющих) и `^?` для DEL.
  - `-t` — отображение табуляций как `^I` и непечатаемых символов.
  - `-e` — отображение символа `$` в конце каждой строки и непечатаемых символов.
  - `-T` — отображение табуляций как `^I`.
  - `-E` — отображение символа `$` в конце каждой строки.
- Поддержка длинных опций (`--number`, `--number-nonblank`, `--squeeze-blank`).

### grep

Реализация `grep` поддерживает:
- Поиск по регулярным выражениям (POSIX Extended Regular Expressions).
- Поддержка множества флагов:
  - `-e` — указание шаблона через аргумент.
  - `-i` — игнорирование регистра.
  - `-v` — инвертирование поиска (вывод строк, не содержащих шаблон).
  - `-c` — вывод только количества совпадающих строк.
  - `-l` — вывод только имён файлов, содержащих совпадения.
  - `-n` — вывод номеров строк.
  - `-h` — скрытие имён файлов в выводе.
  - `-s` — подавление сообщений об ошибках (например, если файл не найден).
- Обработка нескольких файлов.
- Комбинирование нескольких шаблонов через оператор `|`.

## Сборка

Проект использует простые Makefile для сборки. Для компиляции требуется компилятор `gcc`.

### Сборка cat

Перейдите в директорию `cat/` и выполните:

```bash
cd cat/
make
```

Будет создан исполняемый файл `s21_cat`.

### Сборка grep

Аналогично в директории `grep/`:

```bash
cd grep/
make
```

Будет создан исполняемый файл `s21_grep`.

Очистка скомпилированных файлов:

```bash
make clean
```

## Использование

### Использование cat

Общий синтаксис:

```bash
./s21_cat [OPTIONS] [FILE...]
```

Если файлы не указаны, чтение происходит из стандартного ввода.

Примеры:

```bash
# Вывод содержимого файла
./s21_cat file.txt

# Нумерация всех строк
./s21_cat -n file.txt

# Нумерация только непустых строк
./s21_cat -b file.txt

# Сжатие пустых строк и нумерация
./s21_cat -s -n file.txt

# Отображение непечатаемых символов и табуляций
./s21_cat -t -v file.txt
```

### Использование grep

Общий синтаксис:

```bash
./s21_grep [OPTIONS] PATTERN [FILE...]
```

Если файлы не указаны, чтение происходит из стандартного ввода.

Примеры:

```bash
# Простой поиск
./s21_grep "pattern" file.txt

# Поиск с игнорированием регистра
./s21_grep -i "pattern" file.txt

# Вывод номеров строк
./s21_grep -n "pattern" file.txt

# Инвертированный поиск
./s21_grep -v "pattern" file.txt

# Поиск в нескольких файлах
./s21_grep "pattern" file1.txt file2.txt

# Использование флага -e для нескольких шаблонов
./s21_grep -e "pattern1" -e "pattern2" file.txt
```

## Опции

### Опции cat

| Флаг | Длинная опция | Описание |
|------|---------------|----------|
| `-b` | `--number-nonblank` | Нумеровать только непустые строки |
| `-n` | `--number` | Нумеровать все строки |
| `-s` | `--squeeze-blank` | Сжимать несколько пустых строк в одну |
| `-E` | — | Показывать `$` в конце каждой строки |
| `-T` | — | Показывать табуляции как `^I` |
| `-v` | — | Показывать непечатаемые символы |
| `-t` | — | Эквивалент `-vT` |
| `-e` | — | Эквивалент `-vE` |

### Опции grep

| Флаг | Описание |
|------|----------|
| `-e PATTERN` | Указать шаблон для поиска |
| `-i` | Игнорировать регистр при поиске |
| `-v` | Инвертировать поиск (выводить строки, не содержащие шаблон) |
| `-c` | Выводить только количество совпадающих строк |
| `-l` | Выводить только имена файлов с совпадениями |
| `-n` | Выводить номера строк |
| `-h` | Не выводить имена файлов в выводе |
| `-s` | Подавлять сообщения об ошибках (например, о недоступных файлах) |
